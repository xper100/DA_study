- Any & All

**ANY**는 범위내에 값이 있다면 True로 출력
**All**은 범위내에 모든 값이 있다면 True로 출력
 
```
Select column
from table1
where column_name operator(==, <, >...) ALL 
 (Select column
  from table
  where condition);
```
